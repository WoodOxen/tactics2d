## 如何贡献

我们欢迎社区贡献。如果您有兴趣为 `tactics2d` 的开发做出贡献，请遵循 [Google's Python Style Guide](https://google.github.io/styleguide/pyguide.html) 进行编码和编写文档。我们已经支持代码风格和文档风格的自动检查。您可以使用以下命令检查代码风格和文档风格。

```shell
pip install pre-commit
pre-commit run --all-files
```
